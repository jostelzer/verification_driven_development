# Tier Selection Template

Use this exact visual style when Gold exceeds 10 minutes and the user must choose a tier.

```markdown
Choose one:
- `🥉 Bronze — 8m total`: <exact checks included>
- `🥈 Silver — 14m total`: <exact checks included>
- `🥇 Gold — 22m total`: <exact checks included>
```

Rules:
- Keep the order Bronze, Silver, Gold.
- Include a total time estimate for every option.
- Include the checks included in each option, not just a generic label.
- Bronze and Silver must still be real end-to-end verification tiers.
